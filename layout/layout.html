float

  • box 1 (고정)
  • box 2 (고정)

float 다음에 오는 요소 레이아웃 이슈

  • 부모 요소에 강제로 높이 값 부여

    반응형 시 자동 높이 설정 못함.

  • 부모 요소에 overflow: hidden 혹은 auto 적용

    해상 요소의 컨텐츠를 박스 외부로 표현 불가하며 auto 적용시 특정 브라우저에서 스크롤 바 표시됨

  • float 요소의 다음에 오는 요소에 clear:both; 적용

    해당 요소 margin-top 적용 어려움.

  • 가장 권장하는 방법으로 float요소의 부모요소에 가상요소를 만들고 clear:both; 적용
